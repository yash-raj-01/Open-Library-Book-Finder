# 📚 Open Library Book Finder

## 📖 Project Overview
The Open Library Book Finder is a React-based web application that allows users to search for books using a public API. Users can explore book details, view cover images, and track their reading progress.

This project is built to practice working with APIs, handling UI states, and creating interactive frontend applications.

---

## 🎯 Purpose
- Practice API integration with real-world data  
- Improve React skills (hooks, state management)  
- Build an interactive and responsive UI  
- Handle edge cases like empty results and loading states  

---

## 🌐 Public API
This project uses the Open Library API:
- Public API: https://openlibrary.org/developers/api



---

## ✨ Features
- 🔍 Search books by title or author  
- 🖼️ Display book covers in a grid layout  
- 📅 Sort books by publish year  
- 🔄 Flip card animation to view book details  
- 🎯 Reading Goal Tracker ("I've read this" counter)  
- ⚠️ Handle empty search results  
- ⏱️ Debounced search for better performance
